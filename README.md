# A股申万行业轮动规律分析（2021-2024）

## 项目背景

A股不同行业在牛市、熊市、震荡市中的表现差异显著。
本项目基于申万一级行业指数，量化分析27个行业在不同市场环境下的超额收益规律，
识别各类行业的轮动特征，为行业配置提供数据支撑。

## 核心结论

| 行业分组 | 上涨期特征 | 下跌期特征 | 代表行业 |
|--------|---------|---------|--------|
| 成长类 | 弹性最大，胜率60%+ | 跌幅同样放大 | 计算机、电气设备、电子 |
| 周期类 | 胜率高、持续性强 | 随大宗商品回落 | 化工、有色金属 |
| 消费类 | 弹性较弱 | 受估值拖累，表现最差 | 食品饮料、家用电器 |
| 防御类 | 上涨期跑输（胜率0.39） | 下跌期抗跌（胜率0.60） | 银行、公用事业 |

**反常识发现**：2021-2024年间消费类并非防御属性，
食品饮料下跌期日超额收益-0.0023、胜率仅0.40，
是下跌期表现最差的行业，白酒估值杀跌是主因。

## 技术栈

- **数据获取**：AkShare API（申万一级行业指数日线，27个行业）
- **数据处理**：Pandas（pct_change、rolling窗口、groupby聚合）
- **可视化**：Seaborn热力图、Matplotlib折线图
- **分析方法**：超额收益计算、市场环境分类、行业胜率统计
